1. Review CSS syntax

คำถาม คำตอบ
1.CSS syntax ประกอบด้วยอะไรบ้าง? CSS syntax ประกอบด้วย
  1. selector(ใช้ที่เลือกตำแหน่งเพื่อกำหนดลักษณะ)
  2. declaration : การประกาศลักษณะประกอบด้วย
    • property = คุณสมบัติ
    • value = ค่าของคุณสมบัตินั้นๆ
2.การเขียน CSS มีกี่แบบอะไรบ้าง? มี 3 แบบ
  1. 1. inline CSS
  2. 2. internal CSS
  3. 3. external CSS
3.selector มีกี่แบบอะไรบ้าง? หลายแบบ ดังนี้
-ID
-Class
-Pseudo element
-Pseudo class
-Combinators
-Attribute selector
4.Combinator มีกี่แบบอะไรบ้าง? หลายแบบ เช่น
- decendant selector (หรือช่องว่าง) พิมพ์โดย a b
- child selector (>) พิมพ์โดย a > b เลือก b ซึ่งเป็น child ของ a
- adjacent sibling selector (+) พิมพ์โดย a + b เลือก b ที่อยู่ล่าง a และติดกับ a เท่านั้น
- general sibling selector (~) พิมพ์โดย a ~ b เลือก b ที่อยู่ล่าง a ทุกอัน

1.property และ value คืออะไร แตกต่างกันอย่างไร? property คือคุณสมบัติที่เราต้องการจะปรับแต่ง เช่น สี ฟอนต์ ความกว้าง ความยาว
value คือค่าของคุณสมบัติ เช่น สีแดง ฟอนต์kanit 100cm
2.Property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง? 5 กลุ่ม ดังนี้
  1. property for border : border, table
  2. property for content : color, size, font, text, list-style-type, Height, Width, background
  3. property for padding : top right buttom left ของ padding
  4. property for margin : top right buttom left ของ mergin
  5. property for elenment : display height width position float alight flex grid
3.Value unit มีกี่ประเภทคืออะไรบ้าง แต่ละประเภทคืออะไร? มี 2 ประเภท คือ

  2. Relative lenghts คือหน่วยที่เป็นหน่วยเชิงสัมพันธ์ โดยไปอย่างค่ากับอีกอย่างนึง เช่น em, ex, ch, rem, vh, vw, vmin, vmax, %


3. Review CSS Value, Grid, Flex, Justify, Align, Position, Z-index

คำถาม คำตอบ
1. Value ใน Display property มีอะไรบ้าง? -inline : ข้อความอยู่บรรทัดเดียวกัน "ไม่สามารถปรับ ค.กว้าง ค.ยาว ค.สูง ได้"
-block : ข้อความอยู่คนละบรรทัดกัน
-content : ทำให้ element นั้นกลายเป็น content "ไม่สามารถปรับ margin, padding, border, background ได้"
-flex
-grid
-inline-block/flex/grid/table
-run-in
2. grid และ flex แตกต่างกันอย่างไร? Flex ใช้เพื่อจัดการ layout 1 มิติ ส่วน Grid ใช้เพื่อจัดการ layout 2 มิติ
โดย Flex มีลักษณะของการวางของต่างๆเป็นแถวเดียว "ไม่แนวตั้งก็แนวนอน"(1มิติ)
ส่วน grid ใช้เมื่อ item ด้านในมีการจัดวางมีลักษณะเป็น column กับ row (2มิติ)
3. Justify และ Align แตกต่างกันอย่างไร? Justify ใช้จัดรูปแบบตามแนวนอน
Align ใช้จัดรูปแบบตามแนวตั้ง
4. position คืออะไรและมีอะไรบ้าง? คือ ตำแหน่งการจัดวางของ CSS ประกอบด้วย
1. Static block จะโดนเลื่อนตามหน้าเว็บเมื่อ scroll เม้า (default)
2. Relative block เลื่อนจากตำแหน่งเดิมตามที่เรากำหนดว่าจะดันจากทางไหน ซ้าย ขวา บน ล่าง
3. Fixed block เลื่อนเกาะไปกับหน้าเว็บเมื่อ scroll เม้า
4. Absolute เลื่อนออกจากมุมซ้ายบนสุดของหน้าจอตามการกำหนดว่าดันจากทาง ซ้าย ขวา บน ล่าง
5. Sticky เป็นการผสมกันระหว่าง relative กับ fixed
6. Z-index เป็นการจัด element บนหน้าจอในรูปแบบ layer บนหน้าจอ โดยค่าน้อยอยู่หลังสุดส่วนค่ามากอยู่หน้า
โดยค่า default ของทุก element มีค่า Z-index = 0
5. Z-index คืออะไร ใช้อย่างไร? เป็นการจัด element บนหน้าจอในรูปแบบ layer บนหน้าจอ โดยค่าน้อยอยู่หลังสุดส่วนค่ามากอยู่หน้า โดยค่า default ของทุก element มีค่า Z-index = 0
