2.ความเหมือนและความแตกต่างระห่าง 2 tag

tag ความเหมือนและความแตกต่าง
hr กับ br hr ใช้สำหรับหรับขีดเส้นกั้น
br ใช้สำหรับขึ้นบรรทัดถัดไป
ul กับ ol ul ใช้ในการสร้าง list แบบเป็น bullet
ol ใช้ในการสร้าง list แบบเป็น ลำดับ
select,checkbox และ radio button สิ่งที่เหมือนกันของทั้ง 3 คำสั่งคือใช้เมื่อต้องการสร้างตัวเลือก
แตกต่างกันตรงที่ ลักษณะ select จะเป็น dropdown list ส่วน
checkbox กับ radio button เป็นที่สำหรับทำเครื่องหมายโดยนอกจากนี้
Checkbox ต่างกับ radio button ตรงที่ checkbox สามารถเลือกได้หลายตัวเลือก
ส่วน radio button เลือกได้เพียง 1 ตัวเลือกเท่านั้น
div กับ span div ต่อ 1 tag กินเนื้อที่ 1 บรรทัด
ส่วนspan ไม่ว่ากี่ tag ก็จะเอาเนื้อหาของเรามารียงไว้บรรทัดเดียวกัน
script กับ style script is used to define a client-side script (JavaScript).
style is used to define style information for an HTML document.
